# Tetris

![](tetris.png)
## Description
This is a copy of the game Tetris. 
##Controls
* Left Arrow-Move left 
* Up Arrow-Rotate 
* Right Arrow-Move Right 
* Down Arrow-Pull Down 
* C-hold piece
* Space Bar- Slam piece
* Esc - Pause Game
## Features
* JavaFX
* Muti-Threading
* Object Oriented Programming
